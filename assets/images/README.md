# Background Image Setup

## Required Image

Please save the library/study room image provided by the user as:

**Filename**: `library_background.jpg`

**Location**: `assets/images/library_background.jpg`

**Description**: A library interior with bookshelves, study tables, and windows - creates a professional academic atmosphere for the login screens.

## Image Requirements

- Format: JPG or PNG
- Recommended size: 1920x1080 or higher
- The image will be used as background for all three login pages (Admin, Student, Teacher)
- A white overlay (85% opacity) will be applied for form readability

## Alternative

If the image is in PNG format, name it `library_background.png` and the code will automatically detect it.

---

**Note**: The image has been added to the login screens with a semi-transparent white overlay to ensure the login forms remain clearly visible and readable.
